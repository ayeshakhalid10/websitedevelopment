Unused Medicine Donation-GTLMedicine

This unused medicine donation project aims to donate medicines which are not used. The unused medicine can be donated for further utilization by a needy person. This drug donation system helps the user to donate unused medicines to NGO. In this medicine donation system, there are three entities namely, Admin, NGO and Member.NGO can register and login using credentials. They can raise a request of appointment which is furthermore approved and scheduled by admin.Members can register and login using credentials. They can donate medicine by providing medicines details and raising request, further it will be approved by an admin, and they will schedule the donating date.
